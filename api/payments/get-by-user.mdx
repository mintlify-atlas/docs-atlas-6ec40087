---
title: 'Get Payments by User'
api: 'GET /api/payments/:userId'
---

Retrieves all payment records for a specific user.

## Path Parameters

<ParamField path="userId" type="string" required>
  The unique identifier of the user whose payments you want to retrieve
  
  Example: `user-12345`
</ParamField>

## Response

Returns an array of payment objects.

<ResponseField name="id" type="number">
  Unique identifier for the payment
</ResponseField>

<ResponseField name="userId" type="string">
  ID of the user who made the payment
</ResponseField>

<ResponseField name="subscriptionId" type="number">
  ID of the associated subscription
</ResponseField>

<ResponseField name="amount" type="number">
  Payment amount
</ResponseField>

<ResponseField name="status" type="string">
  Payment status. Possible values: `pending`, `completed`, `failed`, `refunded`
</ResponseField>

<ResponseField name="method" type="string">
  Payment method used
</ResponseField>

<ResponseField name="createdAt" type="string">
  Timestamp when the payment was created (ISO 8601 format)
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET https://api.devtoolshub.com/api/payments/user-12345
```

```javascript JavaScript
const response = await fetch('https://api.devtoolshub.com/api/payments/user-12345');
const payments = await response.json();
```

```python Python
import requests

response = requests.get('https://api.devtoolshub.com/api/payments/user-12345')
payments = response.json()
```

</CodeGroup>
