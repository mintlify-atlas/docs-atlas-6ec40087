---
title: 'Confirm Payment'
api: 'PATCH /api/payments/:id/confirm'
---

Confirms a payment by updating its status to `completed` and triggers an email notification to the user.

## Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the payment to confirm
  
  Example: `123`
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the payment
</ResponseField>

<ResponseField name="userId" type="string">
  ID of the user who made the payment
</ResponseField>

<ResponseField name="subscriptionId" type="number">
  ID of the associated subscription
</ResponseField>

<ResponseField name="amount" type="number">
  Payment amount
</ResponseField>

<ResponseField name="status" type="string">
  Payment status (will be `completed` after confirmation)
</ResponseField>

<ResponseField name="method" type="string">
  Payment method used
</ResponseField>

<ResponseField name="createdAt" type="string">
  Timestamp when the payment was created (ISO 8601 format)
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X PATCH https://api.devtoolshub.com/api/payments/123/confirm
```

```javascript JavaScript
const response = await fetch('https://api.devtoolshub.com/api/payments/123/confirm', {
  method: 'PATCH'
});

const payment = await response.json();
```

```python Python
import requests

response = requests.patch('https://api.devtoolshub.com/api/payments/123/confirm')
payment = response.json()
```

</CodeGroup>

## Example Response

```json
{
  "id": 123,
  "userId": "user-12345",
  "subscriptionId": 1,
  "amount": 99.99,
  "status": "completed",
  "method": "credit_card",
  "createdAt": "2026-03-05T10:30:00.000Z"
}
```

<Note>
  This endpoint automatically triggers a payment confirmation email to the user's registered email address. The email includes the payment amount and payment method.
</Note>

## Error Responses

<ResponseField name="404 Not Found" type="error">
  Returned when the payment ID does not exist
  
  ```json
  {
    "statusCode": 404,
    "message": "Payment not found"
  }
  ```
</ResponseField>
