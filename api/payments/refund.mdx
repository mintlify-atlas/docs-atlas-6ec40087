---
title: 'Refund Payment'
api: 'PATCH /api/payments/:id/refund'
---

Refunds a payment by updating its status to `refunded`.

## Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the payment to refund
  
  Example: `123`
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the payment
</ResponseField>

<ResponseField name="userId" type="string">
  ID of the user who made the payment
</ResponseField>

<ResponseField name="subscriptionId" type="number">
  ID of the associated subscription
</ResponseField>

<ResponseField name="amount" type="number">
  Payment amount that was refunded
</ResponseField>

<ResponseField name="status" type="string">
  Payment status (will be `refunded` after processing)
</ResponseField>

<ResponseField name="method" type="string">
  Payment method that was used for the original payment
</ResponseField>

<ResponseField name="createdAt" type="string">
  Timestamp when the payment was originally created (ISO 8601 format)
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X PATCH https://api.devtoolshub.com/api/payments/123/refund
```

```javascript JavaScript
const response = await fetch('https://api.devtoolshub.com/api/payments/123/refund', {
  method: 'PATCH'
});

const payment = await response.json();
```

```python Python
import requests

response = requests.patch('https://api.devtoolshub.com/api/payments/123/refund')
payment = response.json()
```

</CodeGroup>

## Example Response

```json
{
  "id": 123,
  "userId": "user-12345",
  "subscriptionId": 1,
  "amount": 99.99,
  "status": "refunded",
  "method": "credit_card",
  "createdAt": "2026-03-05T10:30:00.000Z"
}
```

<Info>
  This endpoint only updates the payment status in the database. Actual payment gateway refund processing should be handled separately through your payment provider's API.
</Info>

## Error Responses

<ResponseField name="404 Not Found" type="error">
  Returned when the payment ID does not exist
  
  ```json
  {
    "statusCode": 404,
    "message": "Payment not found"
  }
  ```
</ResponseField>
