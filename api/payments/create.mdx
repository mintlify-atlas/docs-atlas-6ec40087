---
title: 'Create Payment'
api: 'POST /api/payments'
---

Creates a new payment record for a user's subscription.

## Request Body

<ParamField body="userId" type="string" required>
  ID of the user making the payment
  
  Example: `user-12345`
</ParamField>

<ParamField body="subscriptionId" type="number" required>
  ID of the subscription associated with the payment
  
  Example: `1`
</ParamField>

<ParamField body="amount" type="number" required>
  Payment amount (decimal with up to 2 decimal places)
  
  Example: `99.99`
</ParamField>

<ParamField body="method" type="string">
  Payment method used (e.g., credit_card, paypal, etc.)
  
  Example: `credit_card`
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the payment
</ResponseField>

<ResponseField name="userId" type="string">
  ID of the user who made the payment
</ResponseField>

<ResponseField name="subscriptionId" type="number">
  ID of the associated subscription
</ResponseField>

<ResponseField name="amount" type="number">
  Payment amount
</ResponseField>

<ResponseField name="status" type="string">
  Payment status. Possible values: `pending`, `completed`, `failed`, `refunded`
  
  Default: `pending`
</ResponseField>

<ResponseField name="method" type="string">
  Payment method used
</ResponseField>

<ResponseField name="createdAt" type="string">
  Timestamp when the payment was created (ISO 8601 format)
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST https://api.devtoolshub.com/api/payments \
  -H "Content-Type: application/json" \
  -d '{
    "userId": "user-12345",
    "subscriptionId": 1,
    "amount": 99.99,
    "method": "credit_card"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.devtoolshub.com/api/payments', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    userId: 'user-12345',
    subscriptionId: 1,
    amount: 99.99,
    method: 'credit_card'
  })
});

const payment = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.devtoolshub.com/api/payments',
    json={
        'userId': 'user-12345',
        'subscriptionId': 1,
        'amount': 99.99,
        'method': 'credit_card'
    }
)

payment = response.json()
```

</CodeGroup>

## Example Response

```json
{
  "id": 123,
  "userId": "user-12345",
  "subscriptionId": 1,
  "amount": 99.99,
  "status": "pending",
  "method": "credit_card",
  "createdAt": "2026-03-05T10:30:00.000Z"
}
```

<Note>
  New payments are created with a `pending` status. Use the [confirm endpoint](/api/payments/confirm) to mark the payment as completed.
</Note>
