---
title: 'Create Subscription'
api: 'POST /api/subscriptions/create'
description: 'Create a new subscription for a user with a specific plan'
---

## Request

<ParamField body="userId" type="string" required>
  ID of the user creating the subscription
  
  Example: `user-12345`
</ParamField>

<ParamField body="planId" type="number" required>
  ID of the plan to subscribe to
  
  Example: `1`
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the subscription
</ResponseField>

<ResponseField name="userId" type="string">
  ID of the user who owns the subscription
</ResponseField>

<ResponseField name="planId" type="number">
  ID of the subscribed plan
</ResponseField>

<ResponseField name="planName" type="string">
  Name of the subscribed plan
</ResponseField>

<ResponseField name="status" type="string">
  Current status of the subscription
  
  Possible values: `active`, `cancelled`, `expired`
</ResponseField>

<ResponseField name="startDate" type="string">
  ISO 8601 timestamp when the subscription started
</ResponseField>

<ResponseField name="endDate" type="string" optional>
  ISO 8601 timestamp when the subscription ends (if applicable)
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp when the subscription was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp when the subscription was last updated
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -X POST https://api.devtoolshub.com/api/subscriptions/create \
  -H "Content-Type: application/json" \
  -d '{
    "userId": "user-12345",
    "planId": 1
  }'
```

```javascript JavaScript
const response = await fetch('https://api.devtoolshub.com/api/subscriptions/create', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    userId: 'user-12345',
    planId: 1
  })
});

const subscription = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.devtoolshub.com/api/subscriptions/create',
    json={
        'userId': 'user-12345',
        'planId': 1
    }
)

subscription = response.json()
```

</CodeGroup>

### Response Example

```json
{
  "id": 42,
  "userId": "user-12345",
  "planId": 1,
  "planName": "Pro Plan",
  "status": "active",
  "startDate": "2026-03-05T10:30:00.000Z",
  "endDate": "2027-03-05T10:30:00.000Z",
  "createdAt": "2026-03-05T10:30:00.000Z",
  "updatedAt": "2026-03-05T10:30:00.000Z"
}
```

<Note>
  The subscription is automatically set to `active` status upon creation. The `startDate` defaults to the current timestamp.
</Note>
