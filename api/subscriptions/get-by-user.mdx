---
title: 'Get User Subscriptions'
api: 'GET /api/subscriptions/:userId'
description: 'Retrieve all subscriptions for a specific user'
---

## Request

<ParamField path="userId" type="string" required>
  ID of the user whose subscriptions to retrieve
  
  Example: `user-12345`
</ParamField>

## Response

Returns an array of subscription objects.

<ResponseField name="id" type="number">
  Unique identifier for the subscription
</ResponseField>

<ResponseField name="userId" type="string">
  ID of the user who owns the subscription
</ResponseField>

<ResponseField name="planId" type="number">
  ID of the subscribed plan
</ResponseField>

<ResponseField name="planName" type="string">
  Name of the subscribed plan
</ResponseField>

<ResponseField name="status" type="string">
  Current status of the subscription
  
  Possible values: `active`, `cancelled`, `expired`
</ResponseField>

<ResponseField name="startDate" type="string">
  ISO 8601 timestamp when the subscription started
</ResponseField>

<ResponseField name="endDate" type="string" optional>
  ISO 8601 timestamp when the subscription ends (if applicable)
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp when the subscription was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp when the subscription was last updated
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -X GET https://api.devtoolshub.com/api/subscriptions/user-12345
```

```javascript JavaScript
const response = await fetch('https://api.devtoolshub.com/api/subscriptions/user-12345');
const subscriptions = await response.json();
```

```python Python
import requests

response = requests.get('https://api.devtoolshub.com/api/subscriptions/user-12345')
subscriptions = response.json()
```

</CodeGroup>
