---
title: 'Cancel Subscription'
api: 'PATCH /api/subscriptions/:id/cancel'
description: 'Cancel an active subscription'
---

## Request

<ParamField path="id" type="string" required>
  ID of the subscription to cancel
  
  Example: `42`
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the subscription
</ResponseField>

<ResponseField name="userId" type="string">
  ID of the user who owns the subscription
</ResponseField>

<ResponseField name="planId" type="number">
  ID of the subscribed plan
</ResponseField>

<ResponseField name="planName" type="string">
  Name of the subscribed plan
</ResponseField>

<ResponseField name="status" type="string">
  Current status of the subscription (will be `cancelled` after successful cancellation)
  
  Possible values: `active`, `cancelled`, `expired`
</ResponseField>

<ResponseField name="startDate" type="string">
  ISO 8601 timestamp when the subscription started
</ResponseField>

<ResponseField name="endDate" type="string" optional>
  ISO 8601 timestamp when the subscription ends (if applicable)
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp when the subscription was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp when the subscription was last updated
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -X PATCH https://api.devtoolshub.com/api/subscriptions/42/cancel
```

```javascript JavaScript
const response = await fetch('https://api.devtoolshub.com/api/subscriptions/42/cancel', {
  method: 'PATCH'
});

const cancelledSubscription = await response.json();
```

```python Python
import requests

response = requests.patch('https://api.devtoolshub.com/api/subscriptions/42/cancel')
cancelled_subscription = response.json()
```

</CodeGroup>

### Response Example

```json
{
  "id": 42,
  "userId": "user-12345",
  "planId": 1,
  "planName": "Pro Plan",
  "status": "cancelled",
  "startDate": "2026-03-05T10:30:00.000Z",
  "endDate": "2027-03-05T10:30:00.000Z",
  "createdAt": "2026-03-05T10:30:00.000Z",
  "updatedAt": "2026-03-05T12:45:30.000Z"
}
```

<Note>
  Cancelling a subscription changes its status to `cancelled`. The `updatedAt` timestamp is automatically updated to reflect when the cancellation occurred.
</Note>
