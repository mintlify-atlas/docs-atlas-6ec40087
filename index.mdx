# DevTools Hub API

DevTools Hub API is a production-ready backend service built with NestJS that provides a complete subscription and payment management system. Built with clean architecture principles, it offers a modular, maintainable, and scalable foundation for subscription-based applications.

## Quick Start

Get up and running with DevTools Hub API in minutes:

<Steps>
  <Step title="Install Dependencies">
    Clone the repository and install dependencies with pnpm
    ```bash
    git clone https://github.com/carlostajandev/devtools-hub-api.git
    cd devtools-hub-api
    pnpm install
    ```
  </Step>
  
  <Step title="Configure Environment">
    Set up your environment variables for database and email
    ```bash
    cp .env.example .env
    # Edit .env with your configuration
    ```
  </Step>
  
  <Step title="Start the Server">
    Run the development server
    ```bash
    npm run start:dev
    ```
    The API will be available at `http://localhost:3000/api`
  </Step>
</Steps>

## Architecture Overview

DevTools Hub API follows hexagonal architecture (ports and adapters) principles:

- **Domain Layer**: Core business entities and repository contracts
- **Application Layer**: Use cases implementing business logic
- **Infrastructure Layer**: Database persistence, controllers, and external adapters
- **Presentation Layer**: DTOs for validation and data transport

This architecture ensures complete decoupling from frameworks and external dependencies, making the codebase testable, maintainable, and framework-agnostic.
