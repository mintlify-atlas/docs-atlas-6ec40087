# Quickstart Guide

This guide will walk you through making your first API requests to DevTools Hub. You'll learn how to register a user, authenticate, create a plan, subscribe to it, and process a payment.

## Prerequisites

Before you begin, make sure you have:

- DevTools Hub API running locally at `http://localhost:3000`
- A tool to make HTTP requests (cURL, Postman, or similar)
- Basic understanding of REST APIs

<Note>
  All API endpoints are prefixed with `/api`. For example: `http://localhost:3000/api/auth/register`
</Note>

## Make Your First Request

<Steps>
  <Step title="Register a New User">
    Create your first user account by sending a POST request to the registration endpoint.

    ```bash cURL
    curl -X POST http://localhost:3000/api/auth/register \
      -H "Content-Type: application/json" \
      -d '{
        "name": "Juan Pérez",
        "email": "juan.perez@ejemplo.com",
        "password": "MiContraseñaSegura123"
      }'
    ```

    ```javascript JavaScript
    const response = await fetch('http://localhost:3000/api/auth/register', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        name: 'Juan Pérez',
        email: 'juan.perez@ejemplo.com',
        password: 'MiContraseñaSegura123'
      })
    });

    const data = await response.json();
    console.log(data);
    ```

    **Response:**
    ```json
    {
      "success": true,
      "message": "User registered successfully",
      "data": {
        "id": "user-12345",
        "email": "juan.perez@ejemplo.com"
      }
    }
    ```

    <Note>
      Upon registration, a welcome email will be automatically sent to the provided email address.
    </Note>
  </Step>

  <Step title="Login and Get Access Token">
    Authenticate with your credentials to receive an access token.

    ```bash cURL
    curl -X POST http://localhost:3000/api/auth/login \
      -H "Content-Type: application/json" \
      -d '{
        "email": "juan.perez@ejemplo.com",
        "password": "MiContraseñaSegura123"
      }'
    ```

    ```javascript JavaScript
    const response = await fetch('http://localhost:3000/api/auth/login', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        email: 'juan.perez@ejemplo.com',
        password: 'MiContraseñaSegura123'
      })
    });

    const data = await response.json();
    console.log(data);
    ```

    **Response:**
    ```json
    {
      "success": true,
      "message": "Login successful",
      "data": {
        "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
        "user": {
          "id": "user-12345",
          "email": "juan.perez@ejemplo.com",
          "name": "Juan Pérez"
        }
      }
    }
    ```

    <Warning>
      Store the `accessToken` securely. You'll need it for authenticated requests.
    </Warning>
  </Step>

  <Step title="Create a Subscription Plan">
    Create a new subscription plan that users can subscribe to.

    ```bash cURL
    curl -X POST http://localhost:3000/api/plans/create \
      -H "Content-Type: application/json" \
      -d '{
        "name": "Premium",
        "description": "Access to all premium features",
        "price": 99.99,
        "isActive": true
      }'
    ```

    ```javascript JavaScript
    const response = await fetch('http://localhost:3000/api/plans/create', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        name: 'Premium',
        description: 'Access to all premium features',
        price: 99.99,
        isActive: true
      })
    });

    const plan = await response.json();
    console.log(plan);
    ```

    **Response:**
    ```json
    {
      "id": 1,
      "name": "Premium",
      "description": "Access to all premium features",
      "price": 99.99,
      "isActive": true,
      "createdAt": "2026-03-05T14:20:00.000Z",
      "updatedAt": "2026-03-05T14:20:00.000Z"
    }
    ```

    <Note>
      Save the `id` from the response - you'll need it to create a subscription.
    </Note>
  </Step>

  <Step title="Create a Subscription">
    Subscribe a user to the plan you just created.

    ```bash cURL
    curl -X POST http://localhost:3000/api/subscriptions/create \
      -H "Content-Type: application/json" \
      -d '{
        "userId": "user-12345",
        "planId": 1
      }'
    ```

    ```javascript JavaScript
    const response = await fetch('http://localhost:3000/api/subscriptions/create', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        userId: 'user-12345',
        planId: 1
      })
    });

    const subscription = await response.json();
    console.log(subscription);
    ```

    **Response:**
    ```json
    {
      "id": 1,
      "userId": "user-12345",
      "planId": 1,
      "status": "active",
      "startDate": "2026-03-05T14:20:00.000Z",
      "endDate": "2026-04-05T14:20:00.000Z",
      "createdAt": "2026-03-05T14:20:00.000Z"
    }
    ```
  </Step>

  <Step title="Create a Payment">
    Process a payment for the subscription.

    ```bash cURL
    curl -X POST http://localhost:3000/api/payments \
      -H "Content-Type: application/json" \
      -d '{
        "userId": "user-12345",
        "subscriptionId": 1,
        "amount": 99.99,
        "method": "credit_card"
      }'
    ```

    ```javascript JavaScript
    const response = await fetch('http://localhost:3000/api/payments', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        userId: 'user-12345',
        subscriptionId: 1,
        amount: 99.99,
        method: 'credit_card'
      })
    });

    const payment = await response.json();
    console.log(payment);
    ```

    **Response:**
    ```json
    {
      "id": 1,
      "userId": "user-12345",
      "subscriptionId": 1,
      "amount": 99.99,
      "method": "credit_card",
      "status": "pending",
      "createdAt": "2026-03-05T14:20:00.000Z"
    }
    ```
  </Step>

  <Step title="Confirm the Payment">
    Confirm the payment to complete the transaction. This will trigger a confirmation email.

    ```bash cURL
    curl -X PATCH http://localhost:3000/api/payments/1/confirm \
      -H "Content-Type: application/json"
    ```

    ```javascript JavaScript
    const response = await fetch('http://localhost:3000/api/payments/1/confirm', {
      method: 'PATCH',
      headers: {
        'Content-Type': 'application/json'
      }
    });

    const confirmedPayment = await response.json();
    console.log(confirmedPayment);
    ```

    **Response:**
    ```json
    {
      "id": 1,
      "userId": "user-12345",
      "subscriptionId": 1,
      "amount": 99.99,
      "method": "credit_card",
      "status": "confirmed",
      "confirmedAt": "2026-03-05T14:25:00.000Z",
      "createdAt": "2026-03-05T14:20:00.000Z"
    }
    ```

    <Note>
      A confirmation email will be automatically sent to the user's email address with the payment details.
    </Note>
  </Step>
</Steps>
